# Homelab
2026-06-14 Sunday

___

A security-focused homelab built to develop practical skills in:

- Linux Administration
- Networking
- Docker
- Self-Hosting
- DevOps
- DevSecOps
- Monitoring
- Infrastructure Documentation

---

## Goals

This project prioritizes:

- Learning over convenience
- Security over convenience
- Reproducibility
- Documentation
- Operational discipline

The homelab serves as both a learning platform and a portfolio project.

---

## Current Infrastructure

### Production Environment

A dedicated Ubuntu Server responsible for stable workloads and externally accessible services.

### Lab Environment

A separate system used for experimentation, testing, and validation before production deployment.
