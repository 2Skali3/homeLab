# Backup Strategy

## Status

Planned

# Objectives

- Minecraft world data
- Server configuration
- Documentation

# Backup Media

Dedicated external 500GB HDD.

Connected through USB enclosure.

---

# Backup Frequency

## Minecraft

Hourly

## System Configuration

Daily

## Documentation

Daily

---

# Retention Policy

### Hourly

24 copies

### Daily

7 copies

### Weekly

4 copies

---

# Recovery Goal

Recover from:

- SSD failure
- Accidental deletion
- Configuration mistakes
- World corruption

without permanent loss of critical data.
