# Architecture Decision Records (ADRs)

This folder will contain ADRs documenting significant design decisions
for the homeLab project — what was decided, why, what alternatives were
considered, and what trade-offs were accepted.

Naming convention: `ADR-001-short-description.md` (zero-padded, kebab-case).

No ADRs exist yet. The first is expected during Phase 1 (Backup and Recovery).
